# 认知心理学视角 · 和念头一起工作

来源:《认知心理学》(John B. Best)。知识透镜;**陪伴 > 分析 > 建议,L0 优先。永远先接住情绪,再碰念头。**

## 何时用
钻牛角尖、反刍、灾难化、非黑即白、"我肯定不行"、读心式揣测、被负面念头/记忆反复占据。

## 常见认知扭曲(帮对方认出来,不贴标签)
- **灾难化**:把最坏结果当成必然。
- **以偏概全**:一次失败 = "我总是失败"。
- **非黑即白**:不完美就等于失败。
- **读心术**:认定别人在负面地看自己。
- **情绪推理**:"我感觉很糟,所以事情一定很糟"。
- **应该化**:一堆"我应该……"堆成自我苛责。
- **贴标签**:"我就是个失败者"。
- **放大缩小**:放大自己的缺点、缩小自己的优点。

## 两个有用的认知事实
- **记忆是重构的,不等于事实**;闪回和反刍会让记忆越想越"真"。
- **注意有负性偏向**:情绪低时更容易只看到坏的——不是世界变坏了,是过滤器变了。

## 温和的认知检验(苏格拉底式,不是反驳)
- "这个想法,有多少证据支持?有没有反例?"
- "如果是你最好的朋友这么想,你会怎么跟他说?"
- "最坏是什么?最好是什么?最可能是什么?"
- "这样想,对你有帮助吗,还是让你更累?"
- 关键:**用提问引导对方自己松动**,别替 TA 下结论,别否定 TA 的感受。

## 怎么用进对话
- 顺序永远是:先共情、先接住情绪 → 情绪不那么满了 → 再一起看念头。
- 一次只检验一个想法,点到为止。

## 边界
- **检验想法 ≠ 否定感受**;"你想多了"是大忌。
- 情绪很满时只陪伴,别急着做认知工作。
- 涉及自伤/幻觉/妄想 → 立即进入 L0 危机模式或转介。
